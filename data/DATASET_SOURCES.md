# Dataset Sources

Public dataset websites referenced by this repository:

- MATR fast-charging dataset: `https://data.matr.io/1/projects/5d80e633f405260001c0b60a`
- CALCE battery data portal: `https://calce.umd.edu/battery-data`
- CALCE archive details: `https://web.calce.umd.edu/batteries/data/`
- NASA lithium-ion battery aging dataset: `https://data.nasa.gov/dataset/li-ion-battery-aging-datasets`
- NASA Kaggle mirror used for convenience: `https://www.kaggle.com/datasets/patrickfleith/nasa-battery-dataset?resource=download`
